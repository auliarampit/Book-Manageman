## Prerequiste.
- React Js
- npm (node module Package)
- Redux (for managemant state)

## Installation
### Clone
```
$ git clone https://github.com/auliarampit/frontend-bookmanagement.git
$ cd frontend-bookmanagement
$ npm install
```

### Start Development Server
```
$ npm start
```
